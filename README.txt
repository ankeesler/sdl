SDL 01 - Network Simulator.
  - A simulated MAC+PHY with easy to use APIs like sdlTransmit() and sdlReceive().
  - Unique 4-byte interface ids for each simulated node.
  - CSMA support.
  - CCA support.
  - Configurable network parameters like MTU, packet lifetime, max hosts, and bandwidth.
